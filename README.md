**Leader trend failure analysis** — examined 467 leader-trend episodes, discovered MA breakdown is a bottoming signal (Spearman r = +0.149, p = 0.009), not a deterioration signal. Post-failure recovery varies by industry: Telecom +37.3% vs Pharma -15.3%.
