**A 股多市值层因子研究** — 微盘（13 亿）到超大市值（2.7 万亿），系统性检验因子在不同市值层的有效性。核心发现：因子方向随市值系统性翻转——教科书逻辑在微盘池中全部反向。沉淀了一套可复用的七层交叉验证方法论。

**龙头趋势失效分析** — 467 段龙头趋势期，发现 MA 跌破是见底信号而非恶化信号（Spearman r = +0.149, p = 0.009）。失效后路径高度依赖行业：通信 +37.3% vs 医药生物 -15.3%。
